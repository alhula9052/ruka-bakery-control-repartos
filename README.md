# Ruka Bakery - Control de Repartos

Prototipo web autocontenido para control de rendiciones de repartidores.

## Cómo usar

1. Abrir esta carpeta en VS Code.
2. Abrir `index.html` directamente en el navegador, o usar la extensión **Live Server**.
3. Iniciar sesión con alguno de los usuarios demo.

## Usuarios demo

- Admin: `admin` / `1234`
- Cajera: `cajera` / `1234`

## Notas

- Todo el CSS y JavaScript está dentro de `index.html`.
- Usa LocalStorage para guardar datos en el navegador.
- No requiere backend ni base de datos.
